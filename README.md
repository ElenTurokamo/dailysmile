# ☀️ Start Your Morning with a Smile

A Telegram bot that sends a warm, uplifting morning message every day at 6:00 AM — powered by Google Gemini AI.

---

## Features

- Daily morning messages generated by Gemini AI
- Personalized, warm tone — like a note from a close friend
- Simple subscribe / unsubscribe flow
- SQLite user storage with automatic deactivation for blocked users
- Configurable send time via environment variables

---

## Commands

| Command | Description |
|---|---|
| `/start` | ☀️ Subscribe to morning messages |
| `/stop` | 😴 Unsubscribe from morning messages |
| `/status` | 🌤 Check your subscription status |

---

## Project Structure

```
.
├── bot.py          # Telegram bot — command handlers, user registration
├── scheduler.py    # Daily job — AI message generation & broadcast
├── users.db        # SQLite database (auto-created on first run)
└── .env            # Environment variables (see below)
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/morning-smile-bot.git
cd morning-smile-bot
```

### 2. Install dependencies

```bash
pip install python-telegram-bot google-generativeai schedule
```

### 3. Configure environment variables

Create a `.env` file or export the variables directly:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
AI_API_KEY=your_google_gemini_api_key
SEND_TIME=06:00          # optional, default: 06:00
DB_PATH=users.db         # optional, default: users.db
RUN_ON_START=false       # optional, set true to fire immediately on launch
```

> Get your Telegram bot token from [@BotFather](https://t.me/BotFather).  
> Get your Gemini API key from [Google AI Studio](https://aistudio.google.com/).

### 4. Run the bot and scheduler

In two separate terminals:

```bash
# Terminal 1 — bot
python bot.py

# Terminal 2 — scheduler
python scheduler.py
```

---

## How It Works

1. A user sends `/start` → saved to SQLite as active.
2. Every day at the configured time, `scheduler.py` calls Gemini AI to generate a short morning message.
3. The message is broadcast to all active subscribers.
4. If a user has blocked the bot, they are automatically deactivated — no retries, no noise.

---

## Environment Variables Reference

| Variable | Required | Default | Description |
|---|---|---|---|
| `TELEGRAM_BOT_TOKEN` | ✅ | — | Token from @BotFather |
| `AI_API_KEY` | ✅ | — | Google Gemini API key |
| `SEND_TIME` | ❌ | `06:00` | Daily send time (HH:MM, 24h) |
| `DB_PATH` | ❌ | `users.db` | Path to SQLite database file |
| `RUN_ON_START` | ❌ | `false` | Send a message immediately on scheduler start |
